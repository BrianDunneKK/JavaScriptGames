Piano - Stage 3

This is Stage 3 in the development of Piano. Stage 1 and Stage 2 used lots of audio files (one for each key) to play every note. In this stage, we stop using the audio files and instead each note is generated using code. We'll see in the next stage that this gives much greater flexibility in what the note sounds like.

We use the JavaScript framework called Tone.js to generate the sounds.

In this stage, these improvements have been made:

  • Load the Tone.JS framework by adding this code to piano.html: 
    <script src="https://unpkg.com/tone"></script>
  • Modify piano.html so that sharp keys use "#" instead of "s", so for example "C#" replaces "Cs". This is how notes are written for Tone.JS.
  • Change from playing the note on "mouseover" back on a mouse "click".
  • In Tone.JS, sounds a produced by a synthesiser, so add the following code to create one:
    • const synth = new Tone.Synth().toDestination();
  • Change the playSound() function to contain:
    • let note = key.getAttribute('data-note');
synth.triggerAttackRelease(note, "4n");

The function triggerAttackRelease is used to start playing a note (for example the note "C#3") and to stop playing it after the duration "4n", which is a 1/4 note (crotchet). So Tone.JS uses "attack" to mean start and "release" to mean stop.
