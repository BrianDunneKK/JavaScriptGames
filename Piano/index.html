Piano - Stage 1

This is Stage 1 in the development of Piano. In this stage, these improvements have been made:

  • Renamed the files to piano.html, piano.css and piano.js, as this is the convention I use on all games.
  • Found sound files for all piano keys online and put them in the Sounds directory
  • Changed the data-note attributes each key in piano.html to match the sound files I added.
  • Changed the directory path to the sound files in piano.js.
  • Removed all ".piano .key:nth-child" styling from piano.css so that there are no unwanted gaps.
  • Removed "position: absolute;" from the black CSS style so that the black keys are in the correct position.
