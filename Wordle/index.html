Wordle - Stage 1

This is Stage 1 in the development of Wordle. This stage is just the user interface of the game. There is no game or UI logic and so there is no JavaScript.

Try ...

  1. Create your own styles.
  2. Add some graphics to the page.

Wordle - Stage 2

This is Stage 2 in the development of Wordle. In this stage, these improvements have been made:

  • A simple Wordle JavaScript app (wordle_app.js) is added. This is loaded by the HTML file.
  • The sample words are removed from the HTML file.
  • A JavaScript game class is added (wordle_game.js) but it has no logic yet.
  • The JavaScript UI class is added to display previous guesses, with colouring, and the latest guess.

  1. To make it better!

Wordle - Stage 3

This is Stage 3 in the development of Wordle. In this stage, these improvements have been made:

  • The game model now handles user input and calculates the letter colours.
  • Sample data is removed from the game and UI files.
  • There is no chnage to the wordle.html, wordle.css or wordle_app.js files.

Wordle - Stage 4

This is Stage 4 in the development of Wordle. In this stage, these improvements have been made:

  • The game model determines if you have won or lost.
  • The UI displays a message depending on whether you've won or lost.
  • A list of common words and a lkist of all allowed words are loaded.
  • Select the secret word from the common word list.
